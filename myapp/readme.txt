this file is for windows users ONLY

To run goapp you must have go and python2.7 installed.

place _runGoApp.bat in your working directory, this is where you are 
editing your .go files.

right click and edit your _runGoApp

edit lines 18-20 to your specific paths for Go, go_appengine, and Python27

save and close.

doubble-click _runGoApp

run the following commands to ensure the path has been correctly modified.
>> go
>> goapp
>> python

if you recive an error, check that your folder paths are correct.
